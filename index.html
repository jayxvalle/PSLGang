Journey of the mzML Parser & GUI Pipeline

From our first sketches to a working pipeline with an interactive GUI, this page captures the milestones that shaped the project and the visuals that guided each step.

01What We Built

Parsing Pipeline

Extracts MS level 1 spectra from .mzML files and structures the results into JSON for downstream analysis.

Interactive GUI

Guides users through file selection, chemical targeting, and graph generation with a clean PyQt5 interface.

Visualization

Provides quick plots of extracted spectra and offers export-ready views that document the experiment flow.
