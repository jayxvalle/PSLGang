05Project Documentation

Background

The Chemical Analysis Instrumentation Lab partnered with PSL to streamline mass spectrometry analysis for water quality research.

Scope & Goals

Deliver a fast, reliable pipeline for MS1 spectra: parse .mzML, visualize KMD calculations, and keep the GUI intuitive.

Known Gaps

Only one chemical compound is currently supported and slope calculations need more validation; x-value ranges still need tuning.

Future Work

Automated moving guide lines on graphs, API integrations with tools like MZmine and SIRIUS, and expanding supported compounds.
